# Cuboid Maker

A simple C program that draws an ASCII cuboid based on a length, width, and height entered by the user.

## How It Works

The program takes three inputs (length, width, height) and prints a 3D-style cuboid using `o` characters for the top and side faces, and `.` characters for the front face. Spacing is calculated to give the shape a proper isometric look.

## Requirements

- GCC (or any C compiler)

## Build and Run

```bash
gcc cuboid.c -o cuboid
./cuboid
```

## Usage

After running the program, you will be prompted for three values:

```
Enter desired length: 5
Enter desired width: 3
Enter desired height: 4
```

The program then prints the cuboid to the terminal.
